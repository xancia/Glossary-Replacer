Glossary Pre-Translate Replacer

Replacement flow: global glossary by allowed domains, then matching local glossary for a book/page. Local glossary overrides global on conflicts.

Changes apply to open tabs automatically.

Global

Replacements only run when URL matches these patterns. One pattern per line, e.g. https://69shuba.tw/*.

Local Glossary (Per Book/URL)

Add entries like https://69shuba.tw/read/382455/*. If a local and global rule share the same source term, local wins.

Supported separators: =, ->, =>, →. Comment lines starting with #, //, or ; are ignored.
